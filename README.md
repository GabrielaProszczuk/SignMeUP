# SignMeUp

The aim of the project is to implement an online platform for creating a personalized timetable according to a template created by the dean's office. Users have the option of assigning a point priority to specific classes, and the algorithm assigns the best possible plan to all students. Additionally, it is possible to replace already assigned classes via the exchange application.
